## Capstones

### 1. Engineering Knowledge Assistant

A RAG assistant for engineering teams that answers questions from docs, code comments, runbooks, and incidents with citations and confidence handling.

Best for: AI Engineer, AI Platform Engineer, FDE.

### 2. Autonomous PR Review Agent

An agent that reviews pull requests, checks repository context, identifies risks, proposes comments, and requires human approval before posting.

Best for: Agent Engineer, Applied AI Engineer.

### 3. Agentic Business Workflow Copilot

A workflow assistant that gathers context, calls tools, updates business records, asks for approval, and records an audit trail.

Best for: FDE, Applied AI Engineer, solutions-oriented AI roles.

## Project Documentation Standard

Every serious project should include:

- Problem statement
- User workflow
- Architecture diagram
- Data flow
- Prompt and tool specs
- Eval methodology
- Observability plan
- Security risks
- Known limitations
- Demo script

Use the templates in [templates/](./templates).
