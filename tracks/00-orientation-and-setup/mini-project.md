# Mini-Project: AI Engineering Workspace

## Objective

Build a small TypeScript CLI or web app that calls an LLM and records operational metadata.

## Requirements

- Accept a user prompt.
- Call an LLM provider.
- Print or render the response.
- Log model, latency, token usage, and errors.
- Load secrets from environment variables.
- Include a short README for running the app.

## Suggested Stack

- TypeScript
- Node.js
- AI provider SDK
- SQLite, JSONL, or simple structured logs

## Evaluation

Test:

- Successful request
- Empty prompt
- Missing API key
- Provider error
- Long prompt

## Deliverables

- Source code
- Run instructions
- Example logs
- Build-log entry

## Stretch Goals

- Add request IDs.
- Add a cost estimate.
- Add a minimal web UI.
