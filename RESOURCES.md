## Papers and Essays Worth Reading

- Attention Is All You Need
- Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks
- ReAct: Synergizing Reasoning and Acting in Language Models

Do not try to read every paper deeply before building. For this roadmap, papers are used to sharpen engineering judgment, not to turn you into a model researcher.

## Practice Corpora

Use realistic documents:

- Internal engineering docs from a demo repo
- API documentation
- Support tickets
- Product requirements
- Runbooks
- Pull requests and review comments
- Customer discovery notes

Avoid toy datasets for portfolio projects unless you are using them only for initial smoke tests.

## Resource Quality Bar

Add a resource only if it helps with at least one of these:

- Explaining a concept clearly
- Designing a system
- Building a project
- Evaluating quality
- Operating in production
- Preparing for interviews
